# Paper B novelty and journal strategy

## Fixed claim

Paper B is not a new occupancy estimator, generic identified-set theory, or universal adaptive-control algorithm. Its contribution is a finite ecological modelling layer that separates present-state estimation from intervention-response reportability.

The central question is:

> When do observations sufficient for ecological state estimation remain insufficient for intervention prediction, and what additional observation or experiment can resolve the distinctions that actually change management outcomes?

## Ecological scope

The framework is organized around recurring ecological structures rather than a focal empirical system:

- degraded state versus restoration response;
- invasion detection versus control response;
- population trend versus demographic intervention;
- regime identification versus reversibility;
- structural models that fit current observations but imply different intervention consequences.

## Boundaries

Occupancy estimates ecological state under observation error. Partial identification licenses set-valued conclusions. Model averaging combines uncertain predictions. Adaptive monitoring optimizes learning or precision. Decision theory chooses actions under utilities. Paper B connects these components through a target-relative reporting layer: it determines which ecological response claims are supported, which model distinctions alter those claims, and which experiment can separate the remaining target-relevant classes.

Failure architecture is supporting evidence-design machinery rather than the headline contribution.

## Defensible novelty statement

> Ecological models often identify current state more sharply than intervention consequence. We connect finite ecological worlds, complete experiment records, set-valued target reporting, target-relative model equivalence, and risk-limited experiment selection in one auditable workflow. Deterministic reporting is valid exactly when the target is constant over the compatible class; otherwise the exact report remains set-valued.

## Comparative benchmark

The benchmark compares state-only reporting, full latent-world identification, generic information-gain design, and target-safe risk-limited design. It must test state accuracy versus response reportability, target-irrelevant information expenditure, experiment-choice disagreement, false resolution, ambiguity, cost, and shared-failure ceilings.

## Journal strategy

Primary target: **Ecological Modelling**. The paper should be presented as a systems-level framework connecting state models, alternative mechanisms, intervention prediction, and monitoring design. Formal theorems support internal sharpness but should not dominate the main narrative.

## Revision gate

- retain no focal or private empirical theme as the organizing application;
- develop multiple ecological archetypes in the main text;
- implement explicit target-irrelevant latent dimensions in simulation;
- implement a genuine expected-information-gain comparator;
- generate final figures for state versus response resolution and experiment-choice divergence;
- move proofs and detailed finite algorithms to the supplement;
- complete the bibliography and journal-format build.
