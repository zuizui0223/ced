# CED–MRM bridge theorem roadmap

## Purpose

Determine whether CED and MRM can support one strong manuscript rather than two
adjacent small papers. The decision depends on a genuine common theorem, not on
shared terminology about uncertainty and interventions.

## Proposed common object

A finite ecological evidence problem consists of:

- a finite latent-world set `W`;
- a finite action grammar;
- a declared experiment design `D`, possibly adaptive;
- a deterministic, bounded-support, or probabilistic observation contract;
- a report target `T: W -> Y` or set-valued target;
- optional action costs and false-alert budgets.

Examples of latent worlds:

- CED: presence/exposure coordinates plus failure-factor states;
- MRM: response types or candidate mechanisms;
- combined example: presence coordinates and response type jointly.

## Bridge theorem A — deterministic experiment-induced quotient

For a deterministic design, let `record_D(w)` be the complete record generated by
latent world `w`. Define

```text
w ~_D w' iff record_D(w) = record_D(w').
```

Then:

1. every report computed from the record is constant on `~_D` classes;
2. an exact deterministic report of `T` exists iff `T` is constant on every
   `~_D` class;
3. if `T` is not constant, the exact record-based report is the set
   `{T(w): w in [w]_D}`;
4. the quotient `W / ~_D` is the maximum exact information supplied by design `D`.

This should recover CED's partial panel quotient and MRM's universal versus
set-valued report criterion.

## Bridge theorem B — target-aware minimal safe quotient

Sometimes the full record quotient retains distinctions irrelevant to the future
report target, while current observation must still be preserved.

Define the coarsest equivalence relation that:

- preserves currently observable macrostate;
- preserves the target report under all declared future actions; and
- is stable under action successors.

Prove that its fixed point is the minimal observation-preserving deterministic
candidate-safe quotient.

This should recover MRM's minimal quotient and provide a target-aware refinement of
CED's panel quotient.

## Bridge theorem C — noisy compatible classes

### Bounded-support contract

For observed record `x`, define the compatible latent set

```text
C_D(x) = {w: x is allowed under world w and design D}.
```

A deterministic report is justified iff `T` is constant on `C_D(x)`. Otherwise the
honest report is set-valued.

### Probabilistic contract

With prior and likelihood, observations induce a posterior over latent worlds or
quotient classes. Deterministic resolution requires an explicitly declared
criterion; otherwise retain posterior ambiguity.

This connects MRM's robust and probabilistic updates with CED's observation-error
contracts.

## Bridge theorem D — reliable adaptive refinement

An adaptive policy selects actions based on past records. Its leaves define a
partition of latent worlds. Under imperfect observation, attach to each leaf:

- a false-alert or incorrect-refinement bound;
- a residual compatible target set;
- action cost;
- optional posterior ambiguity.

The optimization problem is not merely shortest identification. It is to minimize
cost or residual ambiguity subject to a declared risk budget.

A full optimal policy theorem is not required for the combined manuscript. A
finite dynamic program on small declared systems or a safe alpha-spending result may
be sufficient if it follows cleanly from the common setup.

## Required finite witnesses

### Witness 1 — presence only

Show that the common quotient recovers the existing CED panel signature and
residual class size.

### Witness 2 — response type only

Show that the common quotient recovers MRM's universal, typed, and set-valued
reports.

### Witness 3 — joint presence and mechanism ambiguity

Construct worlds `(presence_coordinate, response_type)` such that:

- passive monitoring leaves both dimensions ambiguous;
- one detection action resolves presence but not response type;
- one intervention response resolves type only if presence was established;
- shared observation failure can prevent reliable class refinement.

This witness is essential. Without it, the combination remains editorial rather
than mathematical.

## Integrated ecological example

Recommended system: a rare plant–pollinator interaction monitored across sites.

- latent presence: focal pollinator or interaction channel present/absent by site;
- response types: alternative mechanisms for response to floral-trait manipulation
  or competitor removal;
- observations: cameras, eDNA, or visit records;
- failure factors: weather, camera power, observer access, laboratory batch;
- controls: blanks and known-positive trials;
- intervention: floral manipulation or exclusion experiment;
- target report: whether management action has a universal predicted response,
  requires mechanism typing, or remains set-valued.

## Go/no-go tests

### Go for a combined paper if

- theorem A is more than renamed identifiability and provides a useful exact
  quotient interpretation;
- theorem B gives a nontrivial minimality result connecting both repositories;
- the joint witness uses both presence and mechanism dimensions essentially;
- the integrated example supports one coherent introduction and discussion;
- risk-limited observation results can be expressed as reliability of quotient
  refinement rather than a separate statistical section.

### No-go if

- CED and MRM remain disjoint corollaries with separate notation;
- the bridge theorem is only the tautology that identical data cannot distinguish
  worlds;
- no joint witness produces a new conclusion;
- the paper needs two unrelated worked examples;
- the detection formulas dominate the manuscript and obscure the report criterion.

## Development order

1. formalize theorem A in neutral notation;
2. map existing CED and MRM classes into the common world model;
3. prove theorem B or identify the exact additional assumptions needed;
4. implement the three finite witnesses;
5. build the integrated ecological example;
6. reassess whether the combined paper has one title, one abstract, and one central
   figure;
7. only then restructure CED and MRM manuscript materials.

## Decision deadline

Do not spend another open-ended theorem cycle on either repository before completing
steps 1–4. After the joint witness is evaluated, make a final binary decision:

- **combine** into one evidence-and-honest-prediction manuscript; or
- **separate**, with CED as the primary methods paper and MRM retained as a later
  mechanism-reporting paper.
