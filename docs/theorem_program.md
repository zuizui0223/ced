# CED theorem program

## Central question

Finite observational stability is not, by itself, a certificate that no legal
future exterior interaction can change a focal ecological window. CED separates
that negative result from the positive question of what a declared monitoring or
intervention panel identifies exactly.

## Result I — Delayed exposure and no uniform closure horizon

For any fixed finite deterministic system with a finite prefix grammar, the
all-word quotient stabilizes after finitely many refinement rounds. Across a
family in which an exterior reader is enabled only after an unbounded declared
delay, no fixed observation horizon suffices uniformly. In the binary family with
\(m\) exterior coordinates and delay \(H\),

\[
K_{\mathrm{open}}=m+1,
\qquad
\max_i K_{\mathrm{closed},i}=2,
\qquad
H_\star=H+1.
\]

Memory burden and delayed distinguishability are independent axes.

## Result II — Exact partial quotient of a reset panel

Let a panel cover a set \(R(P)\) of exterior read probes and optionally the
intervention probe \(J(P)\). In the binary delayed family, the panel quotient
retains exactly

\[
K(P)=1+|R(P)|+J(P)
\]

bits: focal baseline, covered exterior bits, and response type when intervened.
All remaining coordinates form the residual ambiguity class.

## Result III — Resources are not interchangeable

Each information-bearing probe consumes one fresh trial and crosses the delay
gate. Consequently trial count, total action effort, and maximum temporal depth
produce distinct exact frontiers. In particular no number of shallow trials can
expose an event legal only after a deeper delay.

## Result IV — Common-mode failure diversity

For each required separator set, count the smallest number of declared failure
modes whose union removes it. A panel is robust to \(r\) common-mode failures iff
every separator set requires at least \(r+1\) modes to cover. Repetitions inside
one shared mode do not improve this guarantee.

## Non-claims

CED does not infer a real ecological grammar, delay, reset mechanism, or failure
architecture. The results are finite deterministic statements under a declared
observation and intervention contract.
